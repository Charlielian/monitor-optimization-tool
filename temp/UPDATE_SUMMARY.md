# 更新摘要 - 无数据小区显示功能

## 问题描述

用户反馈：在保障监控中，如果添加了一个 CGI 不存在的小区（数据表中查不到），该小区不会显示在列表中，导致无法知道哪些小区没有数据。

## 解决方案

修改保障监控功能，即使小区 CGI 在数据表中查不到数据，也要显示该小区，只是指标值都为空值（显示为 "-"）。

## 实现效果

### 前端显示

**有数据的小区：**
```
┌─────────┬──────────┬─────────────────┬──────┬──────┬──────┐
│ 场景    │ 小区名   │ CGI             │ 干扰 │ 流量 │ PRB% │
├─────────┼──────────┼─────────────────┼──────┼──────┼──────┤
│ 测试场景│ 正常小区 │ 460-00-123456-7 │ 5.23 │ 12GB │ 75%  │
└─────────┴──────────┴─────────────────┴──────┴──────┴──────┘
```

**无数据的小区：**
```
┌─────────┬────────────────────┬─────────────────┬──────┬──────┬──────┐
│ 场景    │ 小区名             │ CGI             │ 干扰 │ 流量 │ PRB% │
├─────────┼────────────────────┼─────────────────┼──────┼──────┼──────┤
│ 测试场景│ 测试小区 ⚠️ 无数据 │ 460-00-999999-9 │  -   │  -   │  -   │
└─────────┴────────────────────┴─────────────────┴──────┴──────┴──────┘
         ↑ 淡黄色背景（半透明）
```


## 测试方法

### 快速测试

1. 访问场景管理：`http://localhost:5000/scenarios`
2. 添加一个不存在的 CGI：`460-00-999999-9`
3. 访问保障监控：`http://localhost:5000/monitor`
4. 选择该场景，查看是否显示无数据小区
5. 导出 Excel，检查是否包含 "数据状态" 列

### 预期结果

- ✅ 无数据小区显示在列表中
- ✅ 淡黄色背景 + "⚠️ 无数据" 徽章

## 优势

1. **完整性** - 显示所有配置的小区，不遗漏
2. **可见性** - 快速识别哪些小区没有数据
3. **可追溯** - 导出文件包含完整信息
4. **易排查** - 通过标识快速定位问题

## 注意事项

1. **CGI 必填** - 添加小区时必须提供 CGI
2. **排序规则** - 无数据小区排在有数据小区后面
3. **分页影响** - 无数据小区计入总数
4. **统计不含** - 场景统计不包含无数据小区
