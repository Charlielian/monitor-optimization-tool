# 无数据小区功能测试指南

## 测试目的

验证在保障监控中，即使小区 CGI 在数据表中查不到数据，也能正常显示该小区（指标值为空）。

## 测试环境

- Python Flask 应用
- PostgreSQL 数据库
- 浏览器（Chrome/Firefox/Safari）

## 测试步骤

### 测试 1：添加不存在的小区 CGI

**步骤：**

1. 访问场景管理页面：`http://localhost:5000/scenarios`

2. 选择或创建一个测试场景

3. 添加一个不存在的小区：
   ```
   小区ID: TEST-CELL-001
   小区名: 测试小区（无数据）
   CGI: 460-00-999999-9  # 确保这个CGI在数据库中不存在
   制式: 4G
   ```

4. 访问保障监控页面：`http://localhost:5000/monitor`

5. 选择刚才的测试场景，点击"开始监控"

**预期结果：**

- ✅ 4G 小区详细指标表格中显示该小区
- ✅ 该行背景色为淡黄色（半透明）
- ✅ 小区名后显示 "⚠️ 无数据" 徽章
- ✅ 所有指标列显示 "-"
- ✅ 该小区排在有数据的小区后面

**截图示例：**

## 常见问题

### Q1: 为什么无数据小区不显示？

**可能原因：**
- CGI 字段为空
- 场景中没有添加该小区
- 浏览器缓存问题

**解决方法：**
- 确认小区的 CGI 字段已填写
- 刷新页面（Ctrl+F5 强制刷新）
- 检查浏览器控制台是否有错误

### Q2: 导出的 Excel 中没有 "数据状态" 列？

**可能原因：**
- 使用了旧版本的代码
- 导出功能未更新

**解决方法：**
- 确认 `app.py` 中的 `export_monitor_xlsx_full` 函数已更新
- 重启 Flask 应用

### Q3: 无数据小区显示在有数据小区前面？

**可能原因：**
- 排序逻辑错误

**解决方法：**
- 检查 `services/scenario_service.py` 中的排序代码：
  ```python
  all_cells.sort(key=lambda x: (not x["has_data"], -x["max_prb_util"]))
  ```
