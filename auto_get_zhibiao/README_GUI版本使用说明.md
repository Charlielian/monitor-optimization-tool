# 干扰小区提取工具 - GUI版本使用说明

## 📋 简介

`gui_interference_extractor.py` 是一个带图形界面的干扰小区数据提取工具，基于 tkinter 开发，操作简单直观。

## 🚀 快速开始

### 1. 确保依赖文件存在

GUI版本依赖于独立提取器模块，请确保以下文件在同一目录：
- `gui_interference_extractor.py` （GUI主程序）
- `standalone_interference_extractor.py` （核心提取模块）

### 2. 安装依赖

```bash
pip install requests pandas openpyxl lxml pycryptodome pillow
```

**注意**: tkinter 是 Python 自带的，无需额外安装。

### 3. 运行程序

```bash
python gui_interference_extractor.py
```

或者直接双击 `gui_interference_extractor.py` 文件。

### 功能区域说明

#### 1. 登录配置区域
- **用户名**: 输入登录账号
- **密码**: 输入登录密码
- **登录状态**: 显示当前登录状态（未登录/已登录/登录失败）

#### 2. 查询参数区域
- **网络类型**: 选择要提取的网络类型
  - 5G: 只提取5G干扰小区
  - 4G: 只提取4G干扰小区
  - 5G和4G: 同时提取两种网络类型
- **城市**: 输入要查询的城市名称（如"阳江"）
- **查询天数**: 设置查询最近几天的数据（1-30天）
- **只提取干扰小区**: 勾选后只提取标记为干扰的小区，不勾选则提取所有小区
- **查询日期范围**: 自动显示根据天数计算的日期范围

#### 3. 操作按钮
- **登录**: 执行登录操作
- **开始提取**: 开始提取数据（需先登录）
- **停止**: 停止当前操作（开发中）
- **打开输出目录**: 打开数据输出文件夹
- **清空日志**: 清空日志显示区域

#### 4. 进度显示
- **进度条**: 显示当前操作进度
- **状态标签**: 显示当前操作状态

#### 5. 运行日志
- 实时显示程序运行日志
- 不同级别的日志用不同颜色显示：
  - 黑色: 普通信息
  - 绿色: 成功信息
  - 橙色: 警告信息
  - 红色: 错误信息

#### 6. 状态栏
- 显示程序当前状态

## 📝 使用步骤

### 步骤1: 配置登录信息

1. 在"用户名"框中输入你的账号（默认已填写）
2. 在"密码"框中输入你的密码（默认已填写）

### 步骤2: 执行登录

1. 点击"登录"按钮
2. 等待程序连接服务器
3. 根据提示输入图形验证码（如果OCR识别失败）
4. 输入短信验证码（首次登录需要）
5. 等待登录完成，状态显示"已登录"

### 步骤3: 设置查询参数

1. 选择网络类型（5G/4G/5G和4G）
2. 输入城市名称
3. 设置查询天数
4. 选择是否只提取干扰小区

### 步骤4: 开始提取数据

1. 点击"开始提取"按钮
2. 观察进度条和日志输出
3. 等待提取完成
4. 点击"打开输出目录"查看结果文件

## 💡 使用技巧

### 技巧1: 批量提取多个城市

1. 提取第一个城市的数据
2. 修改城市名称
3. 再次点击"开始提取"
4. 重复以上步骤

### 技巧2: 查看历史数据

- 增加"查询天数"可以查看更长时间范围的数据
- 最多支持30天

### 技巧3: 快速重新登录

- 程序会自动保存cookie
- 下次运行时可能直接使用cookie登录，无需输入验证码

### 技巧4: 日志管理

- 日志过多时点击"清空日志"按钮
- 重要信息会用不同颜色标记

## ⚠️ 注意事项

### 登录相关

1. **首次登录**需要输入短信验证码
2. **图形验证码**可能需要手动输入（如果OCR识别失败）
3. **Cookie有效期**通常为几天，过期后需要重新登录
4. **账号密码**会明文显示，注意使用环境安全

### 查询相关

1. **查询时间**不要设置过长，建议不超过30天
2. **网络连接**需要稳定，查询大量数据可能需要几分钟
3. **数据延迟**平台数据通常有1-2天延迟
4. **城市名称**必须准确，如"阳江"而不是"yangjiang"

### 数据保存

1. **输出目录**: `./interference_data_output/`
2. **文件格式**: Excel (.xlsx)
3. **文件命名**: `{网络类型}_干扰小区_{时间戳}.xlsx`
4. **自动覆盖**: 不会覆盖，每次生成新文件

## 🐛 常见问题

### Q1: 程序无法启动？

**可能原因**:
- 缺少依赖模块
- Python版本过低（需要3.6+）

**解决方法**:
```bash
pip install requests pandas openpyxl lxml pycryptodome pillow
```

### Q2: 提示"找不到 standalone_interference_extractor.py"？

**原因**: 缺少核心模块文件

**解决方法**: 确保 `standalone_interference_extractor.py` 和 `gui_interference_extractor.py` 在同一目录

### Q3: 登录失败？

**可能原因**:
- 账号密码错误
- 网络连接问题
- 验证码输入错误

**解决方法**:
- 检查账号密码
- 检查网络连接
- 删除 `./cookies/` 目录重试

### Q4: 查询不到数据？

**可能原因**:
- 日期范围不对
- 城市名称错误
- 平台无该时段数据

**解决方法**:
- 检查日期范围
- 确认城市名称正确
- 尝试其他时间段

### Q5: 程序卡住不动？

**可能原因**:
- 等待用户输入验证码
- 网络请求超时
- 查询数据量过大

**解决方法**:
- 检查是否需要输入验证码
- 检查网络连接
- 减少查询天数

### Q6: 如何查看验证码图片？

**位置**: `./captcha_images/` 目录

**说明**: 如果OCR识别失败，程序会自动保存验证码图片到该目录

## 📊 输出文件说明

### 文件位置
默认保存在 `./interference_data_output/` 目录

### 文件命名
- 5G: `5G_干扰小区_20250114_153045.xlsx`
- 4G: `4G_干扰小区_20250114_153050.xlsx`

### 数据字段

**5G干扰小区**:
- 数据时间、结束时间、CGI、小区名、频段
- 微网格标识、全频段均值、D1均值、D2均值
- 是否干扰小区

**4G干扰小区**:
- 数据时间、结束时间、CGI、小区名、频段
- 微网格标识、系统带宽、平均干扰电平
- 是否干扰小区
