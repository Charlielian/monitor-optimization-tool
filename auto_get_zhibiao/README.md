# 干扰小区数据提取工具集

## 📋 项目简介

这是一套完整的干扰小区数据提取工具，支持从大数据平台自动登录并提取4G/5G干扰小区数据。提供多种使用方式，满足不同场景需求。

## 🎯 核心功能

- ✅ **自动登录**：支持单点登录，自动处理验证码
- ✅ **Cookie复用**：保存登录状态，减少重复登录
- ✅ **即席查询**：自动进入即席查询模块
- ✅ **数据提取**：支持4G/5G干扰小区数据提取
- ✅ **灵活查询**：自定义时间范围、城市等参数
- ✅ **数据过滤**：可选择只提取干扰小区
- ✅ **Excel导出**：自动保存为Excel格式

## 💡 使用示例

### 示例1：GUI方式提取数据

1. 双击 `启动GUI工具.bat`（Windows）或运行 `./启动GUI工具.sh`（macOS/Linux）
2. 输入账号密码，点击"登录"
3. 设置查询参数（网络类型、城市、天数）
4. 点击"开始提取"
5. 等待完成，点击"打开输出目录"查看结果

### 示例2：命令行方式提取数据

```bash
python standalone_interference_extractor.py
```

按提示输入：
- 网络类型（1=5G, 2=4G, 3=同时提取）
- 查询天数（默认7天）
- 城市名称（默认"阳江"）

### 示例3：编程方式调用

```python
from standalone_interference_extractor import quick_extract

# 提取5G干扰小区数据
df, filepath = quick_extract(
    network_type='5G',
    days=7,
    city='阳江'
)

if df is not None:
    print(f"提取成功: {filepath}")
    print(f"数据行数: {len(df)}")
```

## 📊 输出数据

### 输出位置
- 默认目录: `./interference_data_output/`
- 可通过GUI"打开输出目录"按钮快速访问

### 文件格式
- 格式: Excel (.xlsx)
- 命名: `{网络类型}_干扰小区_{时间戳}.xlsx`
- 示例: `5G_干扰小区_20250114_153045.xlsx`

### 数据字段

**5G干扰小区**:
- 数据时间、结束时间、CGI、小区名、频段
- 微网格标识、全频段均值、D1均值、D2均值
- 是否干扰小区

**4G干扰小区**:
- 数据时间、结束时间、CGI、小区名、频段
- 微网格标识、系统带宽、平均干扰电平
- 是否干扰小区

## ⚠️ 注意事项

### 登录相关
1. 首次登录需要输入短信验证码
2. 图形验证码可能需要手动输入
3. Cookie会自动保存，有效期通常几天
4. 账号密码默认已配置，可在脚本中修改

### 查询相关
1. 查询时间不建议超过30天
2. 平台数据通常有1-2天延迟
3. 需要稳定的网络连接
4. 城市名称必须准确（如"阳江"）

### 环境要求
1. Python 3.6 或更高版本
2. 网络需要能访问 `nqi.gmcc.net:20443`
3. 建议内存 2GB 以上

## 🐛 常见问题

### Q1: 如何选择合适的工具？
- **新手用户**: 使用GUI版本
- **服务器环境**: 使用独立命令行版本
- **快速查询**: 使用快速提取脚本
- **二次开发**: 使用完整功能版本

### Q2: Tesseract OCR必须安装吗？
不是必须的。OCR用于自动识别验证码，不安装也可以手动输入。

### Q3: 如何修改默认账号？
编辑对应脚本文件的配置区域，修改 `DEFAULT_USERNAME` 和 `DEFAULT_PASSWORD`。

### Q4: 数据保存在哪里？
默认保存在 `./interference_data_output/` 目录，可通过GUI"打开输出目录"按钮访问。

### Q5: 如何提取多个城市的数据？
- GUI版本: 依次修改城市名称，多次点击"开始提取"
- 命令行版本: 多次运行脚本，每次输入不同城市
- 编程方式: 使用循环调用
